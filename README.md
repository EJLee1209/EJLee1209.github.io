# 🛠 Tech Stack 🛠

## Android
- **Kotlin** 문법에 대해서 잘 이해하고 있습니다.
- **MVVM** 패턴에 대해 이해하고, 프로젝트에 적용할 수 있습니다.
- 비동기 프로그래밍을 위한 코틀린의 **Coroutine** 에 대해 이해하고 있습니다.
- **Firebase** 와 연동해서 사용할 수 있습니다.
- Retrofit2 를 사용해서 서버와 클라이언트 간의 **통신** 구현을 잘 해낼 수 있습니다.
